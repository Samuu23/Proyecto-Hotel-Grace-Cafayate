Aplicación web para gestionar reservas y administración de un hotel, asegurando disponibilidad real y evitando solapamientos entre fechas.

Funcionalidades
-Validación de disponibilidad de habitaciones y prevención de doble reserva.
-Carga y administración de habitaciones (alta/edición/estado/detalles).
-Gestión de reservas desde un panel de administrador.
-Respuesta a consultas de clientes.
-Métricas para seguimiento y control.
-Gestion de usuarios clientes, operadores y administradores.

Tecnologías
React.js, CSS, Supabase y base de datos relacional (PostgreSQL).
